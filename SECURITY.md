# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it promptly by opening an issue. Please include a description of the vulnerability, the steps to reproduce it, and any other information that can help us to address the issue effectively.

## Response Process

We will review your report and respond as quickly as possible. If your report is valid, we will work to resolve the issue in a timely manner and release a patch or workaround if necessary. Your contributions to ensuring the security of this project are greatly appreciated!
